Projets

API REST en FastAPI

Ce projet consiste en la création d'une API REST complète permettant de gérer un ensemble de Pokémon à partir d'un fichier JSON. J'y ai mis en place une architecture structurée avec des endpoints CRUD, une gestion propre des ressources et une modélisation des données via une dataclass dédiée. L'API intègre également un système de filtrage avancé (types, évolution, statistiques, tri dynamique), ainsi qu'une validation stricte des données grâce à FastAPI et Pydantic. J'ai aussi développé une gestion centralisée des erreurs pour garantir des réponses cohérentes et fiables. Enfin, la documentation automatique générée par FastAPI (Swagger UI / ReDoc) rend l'API intuitive, testable et facile à explorer.

Technologies : Python, FastAPI, JSON

Application web : Checklist

Application web de gestion de checklists développée en HTML, CSS (Tailwind) et JavaScript. Elle permet de créer, organiser et suivre des listes de tâches directement dans le navigateur grâce au stockage local (localStorage). L’interface propose l’ajout de checklists avec titre, description et date d’échéance, ainsi que des fonctionnalités de recherche, filtrage (favoris, importantes, terminées) et modification via des modales. Chaque checklist possède sa propre page dédiée aux tâches, avec un système de statut et de progression. Le projet met en pratique la structuration d’une application front-end, la manipulation du DOM, la gestion de données locales et l’organisation du code en plusieurs scripts.

Technologies : HTML, TailwindCSS, JavaScript
